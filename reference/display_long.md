# Pivot data frame to long format for easy printing

Pivot data frame to long format for easy printing

## Usage

``` r
display_long(df, cols = dplyr::everything())
```

## Arguments

- df:

  data frame

- cols:

  columns to pivot long, default is
  [`dplyr::everything()`](https://dplyr.tidyverse.org/reference/reexports.html)
