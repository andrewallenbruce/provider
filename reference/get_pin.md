# Get a pinned dataset from a [pins](https://pins.rstudio.com/reference/pins-package.html) board

Get a pinned dataset from a
[pins](https://pins.rstudio.com/reference/pins-package.html) board

## Usage

``` r
get_pin(pin, ...)
```

## Arguments

- pin:

  `<chr>` string name of pinned dataset

- ...:

  additional arguments passed to
  [`mount_board()`](https://andrewallenbruce.github.io/provider/reference/mount_board.md)

## Value

`<tibble>` or `<data.frame>`
