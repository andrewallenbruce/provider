List pins from a
[pins](https://pins.rstudio.com/reference/pins-package.html) board

## Usage

``` r
list_pins(...)
```

## Arguments

- ...:

  arguments to pass to
  [`mount_board()`](https://andrewallenbruce.github.io/provider/reference/mount_board.md)

## Value

`<list>` of [pins](https://pins.rstudio.com/reference/pins-package.html)
