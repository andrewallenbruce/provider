# Hospitals 2

Hospitals 2

## Usage

``` r
hospitals2(
  ccn = NULL,
  org_name = NULL,
  city = NULL,
  state = NULL,
  zip = NULL,
  county = NULL,
  hosp_type = NULL,
  ownership = NULL,
  count = FALSE,
  set = FALSE
)
```

## Arguments

- ccn:

  `<int>` CMS Certification Number

- org_name:

  `<chr>` Legal business name

- city, state, zip:

  `<chr>` Location city, state, zip

- county:

  `<chr>`

- hosp_type:

  `<chr>`

- ownership:

  `<chr>`

- count:

  `<lgl>` Return the total row count

- set:

  `<lgl>` Return the entire dataset
