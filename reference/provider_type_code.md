# Provider Type Code Reference Table

Provider Type Code Reference Table

## Usage

``` r
provider_type_code
```

## Format

### `provider_type_code`

A data frame with 108 rows and 3 columns:

- code:

  State code

- abbr:

  State abbreviation

- name:

  State full name
